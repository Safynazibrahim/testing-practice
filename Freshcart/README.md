# FreshCart Testing Project

This project contains my manual testing practice on an e-commerce web application (FreshCart), which I originally developed as a Frontend project.

## Scope

I started by testing the Login functionality, covering different scenarios such as:

* Valid login
* Invalid credentials
* Input validation (empty fields, wrong format)
* Negative test cases

## What I did

* Wrote structured test cases
* Covered positive and negative scenarios
* Focused on validation and user behavior

## Next Steps

* Testing Register functionality
* Exploring Cart and Checkout features
* Reporting bugs using Jira

This project reflects my transition from Frontend Development to Software Testing.
