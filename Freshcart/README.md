# FreshCart Testing Project

This project is a manual testing practice on an e-commerce web application (FreshCart), originally developed as a frontend project.

## Project Objective
The goal of this project is to apply software testing concepts by creating and executing test cases for the login functionality and reporting bugs using Jira.

## Scope of Testing
The initial focus was on the Login feature, covering:

- Valid login scenarios
- Invalid credentials
- Input validation (empty fields, incorrect format)
- Negative test cases

## What I Did
- Designed structured manual test cases
- Executed test scenarios for login functionality
- Identified and reported bugs in Jira
- Added an enhancement suggestion for UI improvement

## Tools Used
- Jira (Bug tracking)
- GitHub (Version control & documentation)
- Manual Testing techniques
